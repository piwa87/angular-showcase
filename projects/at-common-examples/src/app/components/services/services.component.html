In this example, we will fetch all comments from the API and save them in local storage. Our `LocalStorageService` provides several useful methods to manage data efficiently:
  • save(key: string, data: T): void: Saves data to local storage with a specified key.
  • load(key: string): T | null: Loads data from local storage by the specified key. Returns null if the key doesn't exist.
  • delete(key: string): void: Deletes the data associated with the specified key from local storage.
  • hasData(key: string): boolean: Checks if there is data associated with the specified key in local storage.
  • clear(): void: Clears all data from local storage.
